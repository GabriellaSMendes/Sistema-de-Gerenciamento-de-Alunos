# Para rodar o código

## Backend

Criar ambiente virtual:
```
python -m venv .venv
```

Ativar ambiente virtual:
```
.\.venv\Scripts\activate
```

Instalar as bibliotecas necessárias:
```
pip install -r requirements.txt
```

## Frontend
Rodar os comandos:

```
npm create vite@latest frontend
```

```
npm install
```

```
npm install axios
```

```
npm install react-router-dom
```

**Para rodar:**
``` npm run dev```
